# 最新版本说明 - 第五十一轮推进补记 - 2026-04-14

## 本轮补记

- 已把 canvas workspace store 的 bootstrap 从“模块内可配置”推进到“shell session 边界可调用”。
- `canvas-canvas` 新增 bootstrap facade，`AppProviders` 负责认证态接线与退出态 reset。
- 本轮没有伪造远端同步 handler，因此 Step 08 仍维持 `L2`，只把 caller wiring 子项记为 `L3`。

## 已完成

1. 显式 queue/store bootstrap 边界。
2. provider session key 去重，避免 StrictMode 重挂重复接线。
3. reset 时 runtime dispose 与 store reset。
4. Node contract 覆盖 caller wiring 与 runtime 注入边界。

## 仍待完成

1. 真实 `execute(task)` handler。
2. remote ack apply / `remoteCursor` 语义。
3. desktop/background runtime caller。
4. conflict/manual replay UI。
