# Step 08 - 主写入路径接入收口

- 日期：`2026-04-13`
- 波次：`Wave-C / 第四十六轮推进`
- 检查点：`CP08-3 / 主写入路径接入`
- 本轮收口：`permanent-delete + toggleFavorite + clearTrash`

## 交付范围

- 扩展 `@sdkwork/canvas-sync` 的 note operation 集，引入独立 `permanent-delete` 语义。
- 在 `deleteNotePermanently()` 成功路径追加 `note / permanent-delete / queued` 同步任务。
- 在 `toggleFavorite()` 的 clean-save 成功路径追加 `note / upsert / queued` 同步任务，补齐 note 级关键元数据变更。
- 在 `clearTrash()` 成功路径把批量清空废纸篓拆解为多条 `note / permanent-delete / queued` 任务，并要求一次 queue snapshot 持久化完成。
- 不在本轮提前实现 worker、远端回执应用、冲突 UI 或离在线切换 smoke。
