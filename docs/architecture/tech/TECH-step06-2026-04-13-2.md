# Step06-本地恢复入口-2026-04-13

- 日期：`2026-04-13`

## 1. 本轮交付

本轮交付了一条最小但真实可用的 `Step 06` 恢复路径：

1. 本地恢复草稿会在初始化时被读取。
2. 页面可以看到恢复提示。
3. 用户可以打开对应笔记、恢复草稿或放弃草稿。
4. 恢复后的内容会回放到既有编辑器与保存主链。

## 3. 风险控制

1. 恢复候选只允许来自当前工作区 live note，避免 trash / missing note 直接混入恢复面。
2. 恢复动作只回放本地草稿字段，不替换 note identity。
3. 恢复后继续使用现有 `save queue + save feedback + retry policy` 主链，不引入第二套保存状态机。
4. 放弃草稿会同步清理本地恢复检查点，避免伪恢复候选反复出现。
