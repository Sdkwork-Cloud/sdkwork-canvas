## 1. 发布摘要

本轮继续沿 Step 04 的边界收敛路线，将 `createFolder` 写路径从 Store 内嵌编排迁移到 `noteWorkspaceWriteCoordinator.ts`。这是一次工程边界收口，不改变用户可见功能，但会降低后续迭代中的回归风险。

## 2. 代码变更

- `packages/sdkwork-canvas-canvas/src/services/noteWorkspaceWriteCoordinator.ts`
  - 新增 `createFolderState()`
  - 新增 create folder 结果类型

- `packages/sdkwork-canvas-canvas/src/store/useNotesWorkspaceStore.ts`
  - `createFolder()` 改为消费 write coordinator

- `sdkwork-canvas-pc-react/scripts/workspace-write-path.contract.test.mjs`
  - 新增 create folder 写路径 contract
