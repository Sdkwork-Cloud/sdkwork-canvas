# Step 07 顶部搜索与命令面板接入

- 日期: `2026-04-13`
- 轮次: `Wave-B / 第三十七轮推进`
- 关联能力: `CP07-3 / 顶部搜索与命令面板接入`

## 目标

在 `CP07-1` 已冻结搜索文档模型、`CP07-2` 已冻结统一查询 API 的前提下，将工作区顶部搜索与命令面板的 note/folder 候选正式接入 `canvas-search`，结束页面侧分散字符串过滤的状态。

## 接入结果

### 顶部搜索

1. `buildNotesWorkspaceViewModel(...)` 不再只依赖本地 title/snippet/tags 字符串过滤。
2. 工作区搜索改为复用 `buildNotesSearchDocuments(...) + searchNotesSearchDocuments(...)`。
3. 搜索命中范围已扩展到 folder path，顶部搜索可以通过文件夹路径命中 note。
4. `favorites / all / trash` 视图在有查询词时统一走共享搜索结果。
5. `recent` 视图继续保留“按更新时间排序、最多 12 条”的语义，只把匹配集合切到共享搜索结果。

### 命令面板

1. actions / views 仍保持本地静态项，不在本轮改动范围内。
2. note/folder 候选已改为消费共享搜索文档与查询结果。
3. note descriptor 现在携带 folder path 关键词，命令面板搜索可通过文件夹路径命中 note。
4. 非匹配 note/folder 在有查询词时不再继续无差别进入候选集。
