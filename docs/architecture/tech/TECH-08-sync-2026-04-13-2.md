## 交付范围

- 在 `@sdkwork/canvas-sync` 内冻结版本化待同步队列快照协议，明确 `storage key / schema version / queue snapshot / envelope`。
- 提供浏览器侧 queue store，保证待同步任务可以稳定持久化，并在异常 payload、未知 schema 版本下安全降级为空队列。
- 提供重试策略与回放辅助接口，把“自动重试 -> 到点回放 -> 超限失败”从调用侧零散逻辑收束为包内显式 API。
- 保持范围仅在同步包，不提前接入 `canvas-canvas` 主写入路径或同步状态 UI。

- `canvas-sync` 不再只有状态机契约，还具备待同步队列持久化和自动重试的最小可落地边界。
