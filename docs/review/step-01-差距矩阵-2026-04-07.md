# Step 01 差距矩阵

## 1. 说明

本矩阵用于把“当前现实”映射到“目标架构”，并明确后续 Step 归属、风险等级和优先级。

差距等级定义：

- `A 类`：阻塞后续所有重构的结构性问题
- `B 类`：影响主链路正确性和可扩展性的核心问题
- `C 类`：影响交付质量、发布治理和长期维护的问题
- `D 类`：优化类问题，可进入 backlog

## 3. 当前优先级排序

1. `Step 02`：共享壳层与工程骨架收敛
2. `Step 03`：会话安全与配置治理升级
3. `Step 10`：质量门禁与验证环境治理并行预埋
4. `Step 04-06`：初始化链、保存链和本地能力层
5. `Step 07-09`：搜索、同步、桌面可信发布链

## 4. 结论

- 当前最应避免的错误，是在包落点和安全治理还未冻结前，直接推进本地层、同步层和版本历史层
- 当前最优路径仍然是：`02 -> 03 -> 10(预埋) -> 04 -> 05 -> 06`
