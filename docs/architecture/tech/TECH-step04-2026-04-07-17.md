# Step 04 页面命令运行时收敛

- 日期：`2026-04-07`
- 范围：`Step 04 / Wave-B`
- 当前状态：`已完成本轮增量，Step 04 仍为 L3`

## 1. 增量摘要

本轮将页面命令依赖组装、命令执行以及命令面板/对话框命令转换，从 `NotesWorkspacePage.tsx` 中收敛为独立运行时服务 `noteWorkspacePageCommandRuntime.ts`。

## 2. 代码变化

- 新增 `packages/sdkwork-canvas-canvas/src/services/noteWorkspacePageCommandRuntime.ts`
- 更新 `packages/sdkwork-canvas-canvas/src/pages/NotesWorkspacePage.tsx`
- 更新 `packages/sdkwork-canvas-canvas/src/services/index.ts`
- 新增 `scripts/workspace-page-command-runtime.contract.test.mjs`
- 更新 `package.json`
- 更新 `scripts/package-scripts-contract.test.mjs`

## 3. 结果说明

- 页面不再内联命令依赖工厂与执行器的拼装
- 命令面板 action 与对话框确认命令不再由页面直接转换并执行
- 页面容器职责进一步收缩为视图装配、浏览器 ref 适配和少量展示胶水

## 4. 验证结果
