## 风险与限制

- 当前同步卡片反映的是本地 queue state，不是远端 ack state。
- 当前手动 drain 入口只是请求已有 runtime 继续 drain；如果没有注入 runtime，它不会伪造成功。
- 当前仍没有：
  - 真实 `remoteApply`
  - ack apply / `remoteCursor` 合并闭环
  - 真实 conflict recovery UI
  - 离线/在线切换 smoke
